# Logo Design Session — 2026-04-28

## Goal

Design a logo for Career Genie that works as an app icon/favicon.

## Key Decisions

- **Primary color:** Emerald green (user preference)
- **Concept:** A magic lamp whose smoke curls into a question mark shape
  - Lamp represents the "genie" branding
  - Question mark represents the Socratic method approach (the genie's "magic" is asking the right questions)
- **Style:** Minimalist, flat vector, modern SaaS aesthetic — not fantasy or cartoonish
- **Avoided:** Any "wishes" language/framing — the app is a coaching tool, not promising to fulfill wishes

## Design Exploration

### Lamp as progress indicator
- Smoke trail connecting wizard steps, filling in as user progresses
- Each step node: ? (questions) → ⚖️ (ranking) → 💡 (insight)

### Socratic iconography ideas considered
- Thought bubble / question mark emerging from lamp smoke
- Owl motif (classical wisdom symbol)
- Lamp spout doubling as a question mark (strongest concept)
